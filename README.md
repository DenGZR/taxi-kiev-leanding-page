# leanding page

### Uses:
  - gulp
  - webpack
  - CSS : SMACCS + LESS
 
### Install
  You need Gulp installed globally:
```sh
$ npm i -g gulp
$ npm install
# after install run
$ gulp
# open http://localhost:1337/
```
